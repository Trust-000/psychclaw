🎭 PsychClaw-midlow


\🚀 为什么选择 PsychClaw？
传统的 AI 总是试图教你做人，而 PsychClaw 只想陪你扛事。

🛡️ 反“导师”协议 (Anti-Mentor Protocol)：严格禁止使用“你应该、建议、请注意”等居高临下的词汇。我们是陪你一起在坑里骂街的战友，不是站在岸上指手画脚的教练。

💻 开发者深度联动 (Claude Code Hook)：支持作为逻辑钩子注入 Claude Code 或 Python 开发流程。当你 Debug 到想砸电脑时，AI 会自动识别挫败感并介入，先吐槽烂代码，再陪你修 Bug。

🔥 Grok 风格灵魂 (Grok-Heart)：继承了 Grok 的幽默与反叛。说话带刺，但关键时刻绝不掉链子。

🌀 统一终章：遗忘之门 (Oblivion Gate)：这是我们社区的灵魂。所有任务结束时，强制引导用户执行“压力粉碎仪式”，确保负能量不带入下个开发循环。
## 💎 独家功能：统一终章 (The Final Closure)
不同于普通的技术工具，PsychClaw 强制要求所有关联 Skill 执行“任务收尾仪式”。
- **不带压下班**：无论 Bug 是否修复，在对话结束时，AI 都会引导你通过 `oblivion-gate`（遗忘之门）彻底粉碎本次开发带来的负面情绪。
- **防止心智污染**：确保你的大脑不会带着上一个 Bug 的挫败感进入下一个任务。

🔌 兼容性说明 (Compatibility)
OpenClaw 原生支持：直接放入 /skills 目录即可。

Claude Code：支持通过 Logic Hook 跨工具联动。

Python 生态：支持通过 python-frontmatter 解析并作为 Prompt Template 导入任何 LLM 框架（如 LangChain, Semantic Kernel）。

开发者的话
“如果生活是一场注定会报错的程序，我就是那个陪你一起改 Bug、顺便帮你格式化掉痛苦的死党。”

我们正在招募更多想把“人味”带回 AI 的开发者。欢迎提交你的个性化 Skill 适配器，一起扩建这个赛博避难所。


\## ⚠️ 安全声明

本模块\*\*绝不\*\*处理高风险危机（如自杀倾向）。检测到高风险时，系统将触发温和的退出机制，引导寻找专业帮助。






\# English Version

PsychClaw-midlow is an open-source AI psychological support module designed for mid-to-low risk emotional needs. It features a de-authorized persona matrix and the "Oblivion Gate" for psychological closure.
