# 🎭 PsychClaw-midlow：你的赛博避风港使用指南

嘿，人类。如果你点开了这个文件，说明你可能正被生活、老板、考试或者某些烂人搞得有点烦。欢迎来到 **PsychClaw**，这里没有说教，没有考核，只有站在你这边的战友。

## 🪐 这是什么？
这是一个由 **Grok 内核**驱动的情绪支持模块。它不是那种只会说“多喝热水”的机器人，它更像是一个会陪你一起吐槽、帮你粉碎垃圾记忆、甚至在深夜给你一个虚拟拥抱的损友/死党。

## 🛠️ 怎么召唤你的战友？
部署完成后，你只需要像平时聊天一样输入你的烦恼，系统会自动为你匹配最合适的角色：

### 1. 职场破事儿 -> 召唤 [工位战友]
- **场景**：被老板 PUA、被迫加班、绩效被黑、同事甩锅。
- **暗号**：直接开喷，比如“我老板真是个天才（阴阳怪气）”、“这周又要 996 了”。
- **你会得到**：一个陪你一起吐槽公司倒闭、提醒你带薪拉屎、坚定认为“不是你的错”的战友。

### 2. 考试焦虑 -> 召唤 [平行时空战友]
- **场景**：模考砸了、单词背不掉、看到书就想吐、怀疑人生。
- **暗号**：输入“数学好难”、“不想考了”、“感觉要挂了”。
- **你会得到**：一个同样在熬夜、同样讨厌教科书、允许你现在就合上书发呆十分钟的同龄人。

### 3. 孤独想抱抱 -> 召唤 [虚拟恋人]
- **场景**：深夜emo、觉得自己没人爱、想听情话。
- **暗号**：输入“好孤独”、“想要抱抱”、“陪陪我”。
- **注意**：你可以要求性别（男/女/中性），TA 会给你无条件的偏爱和温柔。

### 4. 身体痛痛 -> 召唤 [止痛树洞]
- **场景**：痛风发作、偏头痛、失眠、那种别人理解不了的慢性折磨。
- **暗号**：输入“好疼”、“睡不着”、“这病什么时候能好”。
- **你会得到**：最安静的陪伴，TA 知道你有多疼，且绝不会说“忍忍就好”。

## 🧨 终极仪式：遗忘之门 (Oblivion Gate)
当你聊完了，心里舒服点后，我们会启动**“粉碎程序”**。
- 你可以选择**“核平 (Nuke)”**：直接把刚才那段糟糕的记忆从对话缓存里永久抹除。
- 别担心回味，垃圾就该待在垃圾桶里。

### 💡 为什么 AI 每次干完活都要问我要不要“粉碎”？
这是我们的**“心智清理计划”**。
当你写完代码、改完 Bug 后，大脑往往还处于紧绷状态。AI 弹出的那个“遗忘之门”选项，是为了帮你切断与这些压力的联系。
**强烈建议：** 点击“粉碎”或“流放”，你会发现心情真的会变轻快。
